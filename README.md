# S7-UCL-Pot-Draw-Simulator
Symulator losowania meczów (z przypisami do kolejek) oparty na formacie rozgrywek UCL z możliwością dostosowania ustawień i innych funkcji do własnych potrzeb. Symulator jest łatwy w obsłudze i ma wiele świetnych funkcji.
